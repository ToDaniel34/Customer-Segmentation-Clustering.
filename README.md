# Customer-Segmentation-Clustering.

This notebook applies the K-means clustering algorithm to mall customer data in order to identify distinct groups of customers based on their annual income and spending habits. The dataset includes features such as Age, Gender, Annual Income, and a Spending Score from 1 to 100. The analysis covers exploratory data analysis, univariate clustering, and bivariate clustering, with the ultimate goal of providing a marketing team with data-driven insights to build more targeted and effective campaigns.
